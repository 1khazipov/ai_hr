# Результаты тестирования TTS моделей

## Описание тестов

Тестирование проводилось на 3 больших текстах с английскими словами для сравнения старой и новой реализации TTS.

### Тест 1: Технический текст с терминологией
- **Английских слов:** 14
- **Примеры:** machine learning, deep learning, customer experience, API, cloud computing, frameworks
- **Файл старой модели:** `test_1_old_model.wav` (27.85 сек)

### Тест 2: Бизнес-текст с англицизмами  
- **Английских слов:** 23
- **Примеры:** IT, digital transformation, custom software, enterprise, team, data science, cybersecurity
- **Файл старой модели:** `test_2_old_model.wav` (22.66 сек)

### Тест 3: Научный текст с международными терминами
- **Английских слов:** 18
- **Примеры:** interdisciplinary, Quantum computing, artificial intelligence, cutting-edge, robust algorithms
- **Файл старой модели:** `test_3_old_model.wav` (25.18 сек)

## Проблема со старой моделью

Старая модель (только русская Silero TTS) **полностью пропускает или неправильно произносит английские слова** в тексте, так как она обучена только на русском языке.

## Новая реализация

Новая реализация использует две модели:
- Русская модель Silero TTS для русских слов
- Английская модель Silero TTS для английских слов

Текст автоматически разбивается на сегменты, и каждый сегмент озвучивается соответствующей моделью.

## Как прослушать результаты

Все аудиофайлы сохранены в формате WAV и готовы к прослушиванию. Обратите внимание на то, как английские слова произносятся (или пропускаются) в старой модели.
